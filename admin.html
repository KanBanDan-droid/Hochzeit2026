Firebase Firestore Regeln

Falls der Fehler „Missing or insufficient permissions" erscheint, müssen die Firestore-Regeln angepasst werden:

  1. Öffne console.firebase.google.com
  2. Dein Projekt → Firestore Database → Rules
  3. Ersetze den Inhalt mit den Regeln unten
  4. Publish klicken
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Anyone can write RSVPs (guests submitting forms)
    match /rsvps/{doc} {
      allow create: if true;
      allow read, update, delete: if false;
    }
    // Admin reads all RSVPs — secured by API key in Firebase config
    match /rsvps/{doc} {
      allow read: if true;
    }
  }
}
