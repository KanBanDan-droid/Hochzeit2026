So funktioniert das Laden der Daten

Der Admin liest Daten direkt aus dem Google Sheet als CSV — kein CORS-Problem. Dafür muss das Sheet einmalig öffentlich lesbar gemacht werden:

  1. Google Sheet öffnen
  2. Oben rechts auf „Share" klicken
  3. „Change to anyone with the link" → „Viewer"
  4. Done — Daten werden ab sofort geladen
